# Clementine - Personal Assistant Toolbox

Clementine is a modular web application designed to assist users with various personal tasks, including email refinement, cover letter generation, and meal planning. Built with Next.js, Tailwind CSS, and Shadcn/UI components, this application provides a user-friendly interface and efficient tools to enhance productivity.

## Features

- **Email Refiner**: A tool that allows users to input draft emails and receive refined versions based on optional instructions. This feature helps improve the clarity and professionalism of email communications.

- **Cover Letter Generator**: This tool enables users to generate tailored cover letters by inputting job descriptions. It fetches the "Master Resume" from LocalStorage to create personalized cover letters that stand out.

- **Meal Planner**: Users can input their cravings and daily calorie goals to receive a customized meal plan. The meal plan is presented in a Markdown table format for easy readability and organization.

## Getting Started

To get started with the Clementine project, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd Clementine
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open your browser** and navigate to `http://localhost:3000` to view the application.

## Technologies Used

- **Next.js**: A React framework for building server-rendered applications.
- **Tailwind CSS**: A utility-first CSS framework for styling the application.
- **Lucide React**: A collection of icons for React applications.
- **Shadcn/UI**: A component library for building user interfaces.
